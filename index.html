Bosch Dashboard Suite

Three dashboards, one GitHub-ready deployment path.

This repository contains a live Streamlit application and a static GitHub Pages landing page. The live app hosts the Bosch ticket dashboard, Siemens supplier portal, and Continental supplier portal in one place through Streamlit navigation.

Operations

Bosch Ticket Dashboard

Shows ticket KPIs, status breakdowns, filters, and the shared SQLite-backed records table.

dashboard/app.py
API

Siemens Supplier Portal

Uses the Bosch API flow to submit tickets and review previously submitted API-based records.

dashboard/customer_portal.py
File Drop

Continental Supplier Portal

Simulates the non-API workflow by writing tickets into the mock SFTP drop folder.

dashboard/continental_portal.py

How to deploy from GitHub

  1. Deploy streamlit_app.py on Streamlit Cloud to get the live multi-page app.
  2. Use the files in the pages/ folder as the page wrappers for the three dashboards.
  3. Publish this index.html on GitHub Pages if you want a static launch page in the same repo.
